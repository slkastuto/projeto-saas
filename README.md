# FlowZap

## Visão do Produto
FlowZap é um sistema SaaS para automação e gestão de atendimentos via WhatsApp,
permitindo que empresas criem fluxos automáticos, atendam clientes e centralizem
informações em um único painel.

O foco inicial é simplicidade, estabilidade e entrega rápida de valor.

## Objetivo Inicial
Lançar um bot funcional de WhatsApp que permita:
- Respostas automáticas por palavra-chave
- Atendimento humano via inbox
- Uso de dados personalizados por cliente
- Painel simples e intuitivo

## Público-Alvo Inicial
- Provedores de IPTV
- Prestadores de serviço
- Pequenos e médios negócios
- Empresas que usam WhatsApp como principal canal

## MVP (V1)
Funcionalidades previstas para a primeira versão:
- Login e autenticação
- Dashboard básico
- Inbox de conversas
- Integração com WhatsApp
- Automações simples (palavras-chave)
- Horários de atendimento
- Tags e dados personalizados de clientes

## Roadmap
- V1 (MVP): Bot funcional + painel básico
- V1.1: Construtor visual de fluxo
- V2: Relatórios, métricas e integrações
- V3: IA e automações avançadas

## Arquitetura (Decisões Iniciais)
- Tipo: SaaS multi-tenant
- Backend: a definir
- Frontend: a definir
- Banco de dados: PostgreSQL
- Integração WhatsApp: API não oficial (fase inicial)

## Princípios do Projeto
- Segurança desde o início
- Escalabilidade progressiva
- UX simples e consistente
- Entregar valor antes de complexidade

## Observações Importantes
